# Featured Projects

## BÓVEDA

Financial automation platform designed to improve saving and investing habits by executing predefined financial rules.

Instead of requiring manual decisions, the platform automatically allocates funds based on a user's investment profile.

**Role:** Founder & Backend Developer

I designed the entire backend architecture, domain model and financial automation logic.

### Key Features

**Financial Automation Engine**  
Backend system that allows users to define automated contribution and investment strategies.

**Domain Driven Architecture**  
The platform is implemented using hexagonal architecture separating:

- domain  
- application  
- ports  
- adapters  
- infrastructure  

This keeps the core financial logic isolated from infrastructure and integrations.

**Idempotent Financial Operations**  
Deposit detection and order execution flows are designed to prevent duplicate financial operations.

**Outbox Pattern Processing**  
Financial events are processed using an outbox pattern with scheduled workers to ensure reliable execution.

**Testing First Development**  
Core financial logic is developed with strong test coverage including:

- unit tests  
- integration tests  
- contract tests  

**Resilient External Integrations**  
External services are protected using retry, timeout and circuit breaker strategies with Resilience4j.

### Tech Stack

- PostgreSQL
- Flyway
- JUnit 5

## Aberturas Romano

PVC windows and doors manufacturing business.

The goal of the project was to improve lead conversion by reducing friction in the quotation process.

Many potential clients avoided contacting the company because they did not know the approximate price range for custom windows.

**Role:** Backend Developer & Systems Designer

### Key Features

**Lead Conversion Tool**  
Built a price preview system that allows customers to estimate the cost of their PVC window order before contacting the company.

This significantly reduces friction in the sales process.

**WhatsApp Lead Integration**  
After generating a price estimate, the system allows users to contact the company directly through WhatsApp with the context of their request.

This improves lead qualification and response efficiency.

**Financial Management Backend**  
Implemented backend logic used by the business to manage capital flows and internal financial tracking.

### Business Impact

Providing transparent pricing significantly improved customer conversion by allowing potential clients to understand price ranges before initiating contact.

### Tech Stack

- Java
- Spring Boot
- PostgreSQL
- REST APIs
- Cloud Deployment
